# PICCOLO Weddell Sea Phytoplankton Size

This GitHub repository accompanies the manuscript of Brewin et al. titled “Phytoplankton size structure in the northwestern Weddell Sea”. It includes a Jupyter notebook designed to illustrate how the data were processed, models fitted and figures produced

The notebook can be run through a webpage, just click on the launch-binder image below.
